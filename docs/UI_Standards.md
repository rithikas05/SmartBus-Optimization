# UI Standards Document

This document defines the common UI standards for the entire project.

All team members must follow these standards to ensure a consistent user experience.

# UI Rules

1. Use Angular Material.

2. Follow the common color theme.

3. Use common navbar and sidebar layout.

4. Use Material Tables.

5. Use Material Cards.

6. Use same font throughout the application.

7. Do not create custom themes without approval.

8. Keep UI simple and professional.

---

# Final Objective

The entire application should appear as one unified system and not as separate modules developed by different team members.
