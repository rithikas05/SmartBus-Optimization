# Smart Bus Occupancy Optimization Platform

## Domain
Travel, Transportation & Hospitality

## Objective
Improve bus occupancy optimization and revenue recovery using analytics-driven decision making.

## Technology Stack
- Frontend: Angular
- Backend: Spring Boot
- Database: MySQL

## Team Size
6 Members

## Major Modules
1. Authentication & User Management
2. Bus Management
3. Route & Trip Management
4. Booking Management
5. Waitlist & Seat Reallocation
6. Occupancy Analytics
7. Revenue Analytics
8. Route Performance Monitoring
9. Alert System
10. Recommendation Engine
11. QR-Based Baggage Tracking


Authentication -> auth-module

Bus Route Trip -> bus-route-trip-module
